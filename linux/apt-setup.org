* CPP

Until `gcc` is promoted to an official release, it is necessary to add the test repository:

#+begin_src sh
sudo add-apt-repository -y ppa:ubuntu-toolchain-r/test
#+end_src

Install the latest C++ compiler:

#+begin_src sh
sudo apt install -y gcc-11 g++-11
#+end_src
